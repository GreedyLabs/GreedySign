# GreedySign

PDF 문서에 서명이 필요한 모든 상황을 간편하게 처리할 수 있는 전자서명 서비스입니다.

## 이런 분들께 유용합니다

- 여러 사람의 서명이 필요한 문서를 관리하는 분
- 이메일로 PDF를 주고받으며 서명을 수집하던 분
- 별도 설치 없이 브라우저에서 바로 서명하고 싶은 분

## 사용 방법

**1. 문서 업로드**
PDF를 업로드하면 브라우저에서 바로 확인할 수 있습니다.

**2. 서명자 초대**
서명이 필요한 사람의 이메일을 입력하면 초대 링크가 이메일로 발송됩니다.

**3. 서명 진행**
초대받은 사람은 링크를 클릭하고 Google 계정으로 로그인한 뒤, 원하는 위치에 서명을 배치합니다. 텍스트 입력이나 체크박스도 직접 작성할 수 있습니다.

**4. 완료 확인 및 내보내기**
소유자는 서명자별 진행 상태를 실시간으로 확인할 수 있습니다. 모두 완료되면 서명이 반영된 PDF를 내보낼 수 있습니다.

## 내보내기 방식

| 방식 | 설명 |
|---|---|
| 개인 | 내 서명만 반영된 PDF |
| 합본 | 모든 서명자의 서명이 하나의 PDF에 합쳐짐 |
| 개별 일괄 | 서명자별 PDF를 ZIP 파일로 한번에 다운로드 |
